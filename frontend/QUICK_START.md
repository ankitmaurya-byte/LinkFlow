# Quick Start Guide - urlgram Extension

## 🚀 Load Extension in Firefox

### Step-by-Step

1. **Open Firefox** browser

2. **Navigate to**: `about:debugging`
   - Type this in the address bar and press Enter

3. **Click** "This Firefox" in the left sidebar

4. **Click** "Load Temporary Add-on..." button

5. **Navigate** to this directory:
   ```
   /home/ankit/Public/codes/extension/bookmark-manager
   ```

6. **Select** the file: `manifest.json`

7. **Done!** The urlgram icon should appear in your browser toolbar

## 🎯 First Steps

### Try These Features:

#### 1️⃣ Save Your First Link
1. Go to any website (e.g., github.com)
2. Click the urlgram icon in toolbar
3. Click "+ Save Current Tab"
4. ✓ Link saved with GitHub icon!

#### 2️⃣ Create a Folder
1. Click "New Folder"
2. Name it (e.g., "Dev Resources")
3. Click inside the folder
4. Notice breadcrumbs: All Links / Dev Resources

#### 3️⃣ Search Links
1. Type in the search box
2. See instant filtering

#### 4️⃣ Copy a URL
1. Click on the URL field of any link
2. It copies to clipboard automatically!

#### 5️⃣ Open Playground
1. Click "🔬 Open Playground"
2. Try the AI Research mode
3. Enter any URL and click "Start Research"

#### 6️⃣ View Dashboard
1. Click "Open Dashboard →" in footer
2. See grid/list view toggle
3. Try different sort options

## 📝 Common Actions

### Organize Links
- **Switch tabs**: Click All Links, Work, Personal, or Inspiration
- **Create folder**: Click "New Folder" button
- **Navigate folders**: Click any folder card
- **Go back**: Click breadcrumbs at top

### Manage Links
- **Rename**: Click ⋮ menu → Rename
- **Move**: Click ⋮ menu → Move to...
- **Delete**: Click ⋮ menu → Delete
- **Open**: Click anywhere on link card
- **Copy URL**: Click the URL field

---

## ⚠️ Notes

- Extension loads as **temporary** - it will be removed when Firefox restarts
- To reload: Go back to `about:debugging` and click "Reload"
- Data persists in browser storage even after reload

---

## 🐛 Troubleshooting

**Icon doesn't appear?**
- Make sure you selected manifest.json
- Check for errors in about:debugging console

**Links not saving?**
- Check browser console for errors
- Ensure storage permissions are granted

**Popup not opening?**
- Try reloading the extension
- Check if popup HTML/CSS/JS files are present
